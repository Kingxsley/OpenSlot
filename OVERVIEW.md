# Enjeeoh: free booking and scheduling, built for non-profits

Enjeeoh is a complete booking system for organisations that run appointments,
coaching, mentoring, intake sessions or any kind of one-to-one or group meeting.
It does everything the paid tools do (Calendly, Acuity and the like), with features
those tools charge extra for, and it is free for verified non-profits. No tiers, no
per-seat pricing, no surprise bills.

This page is a plain-language summary of what it does and why it is a good fit.

## Why it suits a non-profit

- Free for verified non-profits. The only requirement is that the organisation is
  genuine, which is confirmed once during sign-up.
- Built for teams. Many staff or volunteers can share one booking page, each keeping
  their own hours.
- Private by design. No advertising, no tracking, sessions are protected, and the
  organisation owns its data and can export or delete it at any time.
- Runs on the organisation's own brand and web address, so it looks like part of their
  site, not a third-party tool.

## What it does

### Bookings the way a service actually works

- Offer a named service (for example "Career Coaching") staffed by several people. The
  public picks the service, sees everyone's combined availability, and books a time.
- Enjeeoh can assign whoever is free, share bookings fairly between staff, or let the
  visitor choose who they see.
- Group sessions and workshops with a set number of seats.
- Waitlists: when a time is full, people can ask to be told if a spot opens, and the
  system emails the next person automatically when someone cancels.
- Intake questions per service, so staff get the context they need before the session.
- Optional rule: one active booking per person, so the same person cannot tie up
  multiple slots. This can apply to a single service or across the whole organisation.

### Right time, right place, for everyone

- Each staff member sets their own hours and their own timezone.
- Visitors always see times in their own local timezone, so there is no confusion.
- Free built-in video calls (a private room is created for every booking), or phone or
  in-person, your choice per service.
- Calendar invites for Google, Outlook and any other calendar.

### Fewer no-shows, less admin

- Automatic email confirmations, plus reminders 24 hours and 4 hours before. (Text
  message reminders are available too if a provider is connected.)
- Every booking email includes a link so the person can reschedule or cancel
  themselves, with no phone calls and no back and forth.
- Staff can ask a client to pick a new time, and the client gets a link to do it.

### Profiles that build trust

- Each staff member can add a photo, a public role or title, and a short bio, so the
  booking page feels personal, like meeting a real person.
- The organisation can show or hide the team list per service.

### See what is happening

- A bookings report shows what was booked, attended, cancelled or rescheduled, and who
  did it, with filters and a spreadsheet export.
- A calendar view shows what is coming up.

## Put it on the organisation's own website

The booking page can live entirely on the organisation's own domain. Two simple ways:

- Embed it in an existing page (for example a "Book" page) with a single line of code.
  The web address never changes, so visitors stay on the organisation's site.
- Or run the whole booking page on the organisation's own domain or subdomain.

If the organisation has its own eligibility rules (for example a programme only for
certain people), those questions can be shown first on their own page, and the booking
only appears once the visitor qualifies.

## Looks like theirs, not ours

- The organisation chooses its own colours and wording for the booking page, with no
  code, and can change it any time.
- Light or dark theme, applied across the whole site.

## Safe and above board

- Strong security throughout: protected sign-in, rate limiting, account lockout after
  repeated failed attempts, and encrypted connections.
- A full set of editable legal documents (Privacy Policy, Terms, Cookie Policy,
  Acceptable Use, and a Data Processing Addendum) are included, with version history so
  changes are tracked.
- Optional two-factor authentication and a configurable password policy.

## What it costs to run

The software itself is free and the organisation keeps full control. The only running
costs are basic hosting and a database, both of which have free or very low-cost tiers
to start with. Email sending is free through common providers.

## In one sentence

A professional, private, team-ready booking platform that lives on the organisation's
own website, removes the manual scheduling work, and costs a non-profit nothing to use.
